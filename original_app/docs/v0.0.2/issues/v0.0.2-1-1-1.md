# Title: v0.0.2-1-1-1 - Make Username Clickable for Profile Navigation

## Summary: 
Convert the static username display in the navigation bar into a clickable link that navigates users to their profile page at the `/profile` route. This enhancement provides an intuitive entry point for users to access their profile information.

### Acceptance Criteria:
- [ ] Username in navbar is displayed as a clickable link (React Router Link component)
- [ ] Link maintains existing display format: `{user.full_name} ({user.role})`
- [ ] Visual hover feedback provided (CSS styling with cursor change and underline)
- [ ] Link navigates to `/profile` route using client-side navigation
- [ ] Navigation works from all pages where navbar is displayed
- [ ] Authentication state is maintained during navigation
- [ ] Link is accessible (keyboard navigable, screen reader friendly)
- [ ] No visual regressions in navbar layout or styling

### Test Strategy:

**Manual Testing:**
- Test clicking username link from customer dashboard
- Test clicking username link from tasker dashboard
- Test clicking username link from task list page
- Test keyboard navigation (Tab to username, Enter to navigate)
- Test hover state visual feedback
- Test authentication persistence after navigation
- Test screen reader announcement of link

**Browser Compatibility:**
- Chrome/Edge (latest)
- Firefox (latest)
- Safari (latest)

**Accessibility Testing:**
- Keyboard navigation test
- Screen reader test (NVDA/JAWS)
- Color contrast verification

**Description:**
Update the Navbar component to convert the static `<span>` element displaying the username into a React Router `<Link>` component. Add CSS styling for hover effects to provide visual feedback. Import the Link component from react-router-dom.

**File:** `app/frontend/src/components/Navbar.js`

**Current Implementation (lines 13-15):**
```jsx
<span>
  {user.full_name} ({user.role})
</span>
```

**Updated Implementation:**
```diff
  import React from 'react';
- import { useNavigate } from 'react-router-dom';
+ import { useNavigate, Link } from 'react-router-dom';

  // ... existing code ...

- <span>
-   {user.full_name} ({user.role})
- </span>
+ <Link to="/profile" style={{ color: 'white', textDecoration: 'none' }}>
+   {user.full_name} ({user.role})
+ </Link>
```

## üìù Code Changes

**Description:**
Add CSS styling for the username link hover state to provide visual feedback when users interact with the link.

**File:** `app/frontend/src/index.css`

```css
/* Username link hover effect in navbar */
.navbar a[href="/profile"] {
  transition: opacity 0.2s ease-in-out;
}

.navbar a[href="/profile"]:hover {
  text-decoration: underline;
  opacity: 0.9;
  cursor: pointer;
}

.navbar a[href="/profile"]:focus {
  outline: 2px solid white;
  outline-offset: 2px;
}
```
