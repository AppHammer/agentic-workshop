# Title: v0.0.2-1-1-1 - Enable Edit Mode

## Summary: 
This task involves implementing the client-side functionality to enable an "edit mode" on the user profile page. When a user clicks the "Edit" button, the static profile fields will transform into editable input fields, and the action buttons will change to "Save" and "Cancel."

### Acceptance Criteria:
- Given the user is viewing their profile page, when they click the "Edit" button, the static profile fields (Email, Skills, Hourly Rate) should become editable input fields.
- The "Edit" button should be replaced by "Save" and "Cancel" buttons.
- The input fields should be populated with the user's current data.

### Test Strategy:
- **Unit Testing:** Use Jest and React Testing Library to test the `Profile.jsx` component.
  - Verify that the component's state correctly transitions from `isEditing = false` to `isEditing = true` on button click.
  - Assert that input fields are rendered with the correct initial values when in edit mode.
  - Confirm that the "Save" and "Cancel" buttons are visible in edit mode.
- **Manual Testing:**
  - Log in and navigate to the profile page.
  - Click the "Edit" button and verify the UI changes as expected.

**Component Name:** `Profile.jsx`

**Description:**
The existing `Profile.jsx` component will be modified to manage an internal `isEditing` state. When `isEditing` is true, the component will render `input` elements for the user's email, skills, and hourly rate, allowing for modification.

**Code:**

```jsx
import React, { useState, useContext } from 'react';
import { AuthContext } from '../AuthContext';
import { api } from '../api';

const Profile = () => {
    const { user, setUser } = useContext(AuthContext);
    const [isEditing, setIsEditing] = useState(false);
    const [formData, setFormData] = useState({
        email: user.email,
        skills: user.skills || '',
        hourly_rate: user.hourly_rate || 0,
    });

    const handleEdit = () => {
        setIsEditing(true);
    };

    // Save and Cancel handlers will be implemented in subsequent tasks.

    if (!user) {
        return <div>Loading...</div>;
    }

    return (
        <div>
            <h2>Profile</h2>
            {isEditing ? (
                <div>
                    <p><strong>Email:</strong> <input type="email" value={formData.email} /></p>
                    <p><strong>Skills:</strong> <input type="text" value={formData.skills} /></p>
                    <p><strong>Hourly Rate:</strong> <input type="number" value={formData.hourly_rate} /></p>
                    <button>Save</button>
                    <button>Cancel</button>
                </div>
            ) : (
                <div>
                    <p><strong>Username:</strong> {user.username}</p>
                    <p><strong>Email:</strong> {user.email}</p>
                    <p><strong>User Type:</strong> {user.user_type}</p>
                    <p><strong>Skills:</strong> {user.skills}</p>
                    <p><strong>Hourly Rate:</strong> ${user.hourly_rate}</p>
                    <button onClick={handleEdit}>Edit</button>
                </div>
            )}
        </div>
    );
};

export default Profile;
```
